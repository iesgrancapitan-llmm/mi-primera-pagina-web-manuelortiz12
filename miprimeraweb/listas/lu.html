Listas no ordenadas


  • W3C

    • Opera como un consorcio donde diversas organizaciones y expertos colaboran para establecer normas y recomendaciones que promueven la interoperabilidad y el crecimiento de la web.

  • WHATWG

    • WHATWG, que significa Web Hypertext Application Technology Working Group, es un grupo de trabajo que se formó en 2004 con el objetivo de desarrollar y mantener tecnologías web relacionadas con hipertexto y aplicaciones

  • HTML

    • HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el estándar utilizado para la creación y el diseño de páginas web. Es un lenguaje de marcado que permite la estructuración de contenido en la web mediante el uso de etiquetas y atributos.
