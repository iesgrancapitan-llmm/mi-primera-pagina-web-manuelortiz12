Listas descriptivas

XHTML
XHTML, que significa Extensible HyperText Markup Language (Lenguaje de Marcado de Hipertexto Extensible), es una versión más estricta y extensible de HTML. Fue desarrollado para abordar algunas de las limitaciones y ambigüedades presentes en las versiones anteriores de HTML y para alinearse más estrechamente con las reglas de XML
HTML
HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el estándar utilizado para la creación y el diseño de páginas web. Es un lenguaje de marcado que permite la estructuración de contenido en la web mediante el uso de etiquetas y atributos.
